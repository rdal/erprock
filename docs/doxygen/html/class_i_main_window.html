Detailed Description

Main Window Interface.

Author:
Rafael Donato <rafaeldonato@gmail.com>

This interface tells what methods must be implemented on all MainWindow's.

virtual void IMainWindow::addFormAddCfopToMDI ( ) [pure virtual]

Adds the form, that will add a cofins, into MDI area.

This is an abstract method that will get called, when an user clicks on "Add cofins" button, located at AddCfopView form. There will be placed a child dialog into MDI to allow the user to add more cofins into database. Only Managers, Supervisors and Stockers are allowed to get into this option.

See also:
